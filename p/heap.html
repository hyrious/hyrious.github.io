堆 (数据结构)

堆 (heap) 是一种树状数据结构, 一般具有维护 O(logn), 查询顶点 O(1) 的复杂度, 通常用来解决调度算法等在线问题. 数据结构课没有把它作为考点, 但是日常还是挺常用的, 在此记录.

调库 Man

  • C++ priority_queue / make_heap push_heap pop_heap
  • Java PriorityQueue
  • Python heapq

实现 (以底层是数组的二叉堆为例)

插入节点

先将待插入元素放到树的末尾, 然后从它开始向上调整 (与父节点交换) 以满足堆的性质. 显然该调整不会超过 O(logn) 次.

对于数组实现, 可以直接取数组长度来索引到末尾位置；而简单链式实现中, 要取到末尾位置还需要 O(logn) 的搜索时间, 因此推荐使用数组实现, 如果要尽量使用少的内存, 可以使用倍增数组, 调整倍增系数来优化用时和空间.

#define ROOT 0
#define LEFT(i)   (2 * (i) + 1)
#define RIGHT(i)  (2 * (i) + 2)
#define PARENT(i) ((i - 1) / 2)

int heap[4096];
int n = ROOT; // 当前元素个数
#define ORDER(p, c) ((p) <= (c))
void heap_up(int i) {
    int p = PARENT(i);
    if (!ORDER(heap[p], heap[i])) {
        swap(p, i);
        heap_up(p);
    }
}

void insert(int value) {
    heap[n++] = value;
    heap_up(n - 1);
}

删除根节点

先将末尾节点塞到根节点, 然后从它开始向下调整以满足堆的性质. 显然复杂度仍为 O(logn).

void heap_down(int i) {
    int l = LEFT(i), r = RIGHT(i), p = i;
    if (l < n && !ORDER(heap[p], heap[l])) p = l;
    if (r < n && !ORDER(heap[p], heap[r])) p = r;
    if (p != i) {
        swap(p, i);
        heap_down(p);
    }
}

int pop(void) {
    int value = heap[ROOT];
    heap[ROOT] = heap[--n];
    heap_down(ROOT);
    return value;
}

构造二叉堆

假设已经有一列无序元素, 将其构造成符合堆的性质. 如果使用 n 次插入算法, 其时间复杂度为 O(nlogn). 而更优的算法是自底向上地对每个子树做上文提到的 heap_down 操作, 其时间复杂度为 O(n).

void build(void) {
    for (int i = PARENT(n); i >= ROOT; --i) {
        heap_down(i);
    }
}

由此可以立刻得到一个 O(nlogn) 的堆排序算法, 而且还是原地的 (空间复杂度 O(1)) , 不再赘述.

合并

简单串联两个堆, 然后执行一遍构造即可, 复杂度为 O(n).

实现 (以可并二叉堆为例)

可并堆可以做到 O(1) 的合并效率, 见 merge 函数.

#define ORDER(p, c) ((p) <= (c))
typedef struct Node Node;
struct Node {               //   p---s
    int value;              //  /
    Node *child, *sibling;  // c
};

Node *merge(Node *a, Node *b) {
    if (!a) return b; if (!b) return a;
    if (!ORDER(a->value, b->value)) swap(a, b);
    b->sibling = a->child; a->child = b;
    return a;
}

Node *insert(Node *root, int value) {
    Node *p = malloc(sizeof(Node));
    *p = (Node) { value, NULL, NULL };
    return merge(root, p);
}

Node *merges(Node *x) {
    if (!x || !x->sibling) return x;
    Node *a = x->sibling, *b = a->sibling;
    x->sibling = a->sibling = NULL;
    return merge(merge(x, a), merges(b));
}

Node *pop(Node *root) {
    Node *p = merges(root->child);
    free(root);
    return p;
}
